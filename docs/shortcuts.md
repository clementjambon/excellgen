# Shortcuts Sheet

| Key                 | Description                                                                                                          |
| ------------------- | -------------------------------------------------------------------------------------------------------------------- |
| f                   | Switch between features and RGB rendering                                                                            |
| q                   | Switch between quantized features and raw features                                                                   |
| Ctrl                | Hold to select segments when rendering **quantized features**. Select again to discard selected segments.            |
| Alt                 | Hold to select Gaussians/points. If you use the wheel while pressing Alt, you can adjust the radius of the selection |
| s (point cloud)     | Switch between add/remove selection mode                                                                             |
| h                   | Hide/show selection (or latent voxels)                                                                               |
| o                   | Pre-visualize selected Gaussians                                                                                    |
| Ctrl + Z/Y          | Undo/Do selection (in Latent Exporter with a PcSelector) or displacement                                             |
| s (latent exporter) | Switch between move+rotate/scale with the bounding box                                                               |
| m                   | Move+rotate layer (with the gizmo)                                                                                   |
| n                   | Scale layer (with the gizmo)                                                                                         |
| 1                   | Focus camera on the currently selected layer                                                                         |
| ↑/↓/→/←             | Move camera front/back/right/left                                                                                    |
| rshift/ rctrl       | Move camera up/down                                                                                                  |
| page_up / page_down | Scale layer up/down (NOTE: this isn't compatible with Ctrl+Z/Y)                                                      |
| a                   | Apply selection or brush                                                                                             |
| r                   | Reset brush painting                                                                                                 |
| g                   | Grow (+ patch automatically)                                                                                         |
| p                   | Patch (will only do something if you manually updated patch parameters)                                              |
| t                   | Screenshot 3D Gaussians only                                                                                         |
| u                   | Screenshot with Polyscope                                                                                            |
